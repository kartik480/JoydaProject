# JoyDa fonts (Baloo 2 & Nunito)

Fonts are **bundled** so the app uses the same font style on all devices (no runtime download).

- `Baloo2-Regular.ttf`, `Baloo2-SemiBold.ttf`, `Baloo2-Bold.ttf` — titles and headings  
- `Nunito-Regular.ttf` — body text  

To re-download (e.g. after deleting): run from project root:

```bash
dart run tool/download_fonts.dart
```
